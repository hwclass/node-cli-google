# node-cli-google
A command line tool to search google

####Install

* Firstly, get [https://docs.npmjs.com/getting-started/installing-node](Node.js and npm).

* Then, install the package,
<pre lang="javascript">
<code>
npm install node-cli-google
</code>
</pre>
