# node-cli-google
A command line tool to search google

####Install

* Firstly, get (Node.js and npm)[https://docs.npmjs.com/getting-started/installing-node].

* Then, install the package,
<pre lang="javascript">
<code>
npm install node-cli-google
</code>
</pre>
